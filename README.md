Lagan core files
================
Contains the Lagan base model for all Lagan models. Each type of content has it's own model that extends this model.
Also contains the Lagan search controller.

To be used with [Lagan](https://github.com/lutsen/lagan). Lagan lets you create flexible content objects with a simple class, and manage them with a web interface.
